# 📖 Contexto: ¿Por qué Python?

## La pregunta importante

Si ya sabes analizar datos en Google Sheets, ¿por qué aprender a programar?

Porque Sheets tiene límites:
- **Volumen:** Solo ves miles de filas. ¿Qué pasa con millones?
- **Repetición:** Si tienes que hacer el mismo análisis 50 veces con datos distintos, ¿vas a hacerlo manual?
- **Complejidad:** Algunas transformaciones son imposibles o muy difíciles sin código.
- **Automatización:** No puedes decirle a Sheets "haz esto cada mañana automáticamente".

Python resuelve todo eso. Y es el lenguaje más usado en análisis de datos a nivel mundial.

## ¿Qué es Python?

Python es un **lenguaje de programación**. Es decir, una forma de darle instrucciones a una computadora.

A diferencia de Sheets (donde haces clic), en Python **escribes** lo que quieres que pase:

```python
# En Sheets: haces clic en "Suma"
# En Python: escribes
total = sum([10, 20, 30, 40])
```

Parece más trabajo, pero es mucho más poderoso.

## ¿Por qué Python y no otro lenguaje?

Porque:
- **Es fácil de leer.** Casi parece inglés.
- **Tiene herramientas increíbles para datos.** Pandas, NumPy, Matplotlib, Scikit-learn...
- **Es el estándar en la industria.** La mayoría de empresas que hacen análisis de datos usan Python.
- **Tiene una comunidad enorme.** Si tienes una duda, alguien ya la tuvo antes y la respondió.

## ¿Qué vamos a aprender exactamente?

En Kepler aprenderás los **fundamentos de Python**:

1. **Variables y tipos de datos** → Cómo guardar información
2. **Operadores** → Cómo hacer cálculos y comparaciones
3. **Condicionales** → Cómo hacer que el código tome decisiones (if/else)
4. **Bucles** → Cómo repetir acciones sin copiar código (for/while)
5. **Funciones** → Cómo organizar tu código en bloques reutilizables
6. **Estructuras de datos** → Cómo trabajar con listas y diccionarios

Esto es lo mínimo necesario para luego trabajar con librerías como Pandas.

## ¿Necesito instalar algo?

Sí. Necesitas:
1. **Python** (el lenguaje en sí)
2. **Un editor de código** (como VS Code, PyCharm, o Jupyter Notebook)

Tu tutora te guiará en la instalación. Si usas **Google Colab**, puedes empezar sin instalar nada (solo necesitas un navegador).

## Tu primer script

Cuando tengas Python instalado, crea un archivo llamado `hola.py` y escribe esto:

```python
print("Hola, soy una analista de datos")
```

Guarda el archivo y ejecútalo. Verás tu mensaje en la pantalla.

**Acabas de programar.** En serio. Eso es código. Y funciona.

## ¿Cómo se ejecuta código Python?

Hay varias formas:
- **Desde la terminal:** `python hola.py`
- **Desde un editor como VS Code:** botón de "Run"
- **Desde Jupyter Notebook:** botón de "ejecutar celda"
- **Desde Google Colab:** igual que Jupyter, pero en el navegador

Tu tutora te mostrará el método que usarás en el bootcamp.

## Una última cosa

Aprender a programar no es como aprender teoría. Es como aprender a tocar un instrumento: necesitas **practicar**.

No basta con leer código. Tienes que escribirlo, ejecutarlo, ver qué pasa, romperlo, arreglarlo.

Cada línea de código que escribas te acerca más a automatizar análisis como los que hacías en Sheets, pero a escala real.
