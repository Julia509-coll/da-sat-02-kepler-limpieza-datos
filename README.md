## ¿Qué es Kepler?

**Kepler es el Proyecto 2 del itinerario de Data Analyst.** Aquí aprenderás los fundamentos de Python aplicados al análisis de datos.

En Atlas trabajaste con datos en Google Sheets. Ahora vas a aprender a **darle instrucciones a una computadora** usando Python: variables, condicionales, bucles, funciones. Todo lo necesario para luego automatizar el análisis de datos.

No vamos a hacer todavía análisis con Pandas ni visualización avanzada. Pero sí vas a entender **cómo funciona la programación** y cómo resolver problemas algorítmicos paso a paso.

Este satélite es tu puerta de entrada al código. Y está diseñado para que no necesites experiencia previa.

### ¿Por qué Python después de Sheets?

En Atlas aprendiste a **pensar analíticamente** con datos. Ahora necesitas aprender a **automatizar ese pensamiento** con código.

Python te permite:
- Procesar miles de filas en segundos (no solo las que ves en pantalla)
- Repetir análisis de forma automática
- Escribir instrucciones claras que cualquier computadora puede ejecutar
- Prepararte para herramientas más avanzadas (Pandas, visualización, machine learning)

## ⚠️ Una nota importante

Si nunca has programado, es **completamente normal** que al principio te sientas perdida.

Ver código por primera vez puede dar vértigo. Puede que te equivoques en la sintaxis mil veces. Puede que un bucle no haga lo que esperabas. **Todo eso es parte del proceso.**

Algunas verdades que necesitas saber:

- **Programar es como aprender un idioma nuevo.** Al principio cuesta, pero de repente "hace clic" y empieza a fluir.
- **Los errores son tus maestros.** Cada error te dice algo. Léelo, entiéndelo, corrígelo.
- **Nadie escribe código perfecto a la primera.** Ni siquiera las personas con años de experiencia.
- **Practicar es obligatorio.** Leer código no te enseña a programar. Escribirlo, sí.
- **Preguntar no es señal de debilidad.** Es señal de que estás aprendiendo activamente.

Python no es difícil. Es diferente. Y tú puedes aprenderlo.

## 💡 ¿Por qué este satélite se llama Kepler?

Johannes Kepler descubrió las leyes del movimiento planetario usando datos astronómicos. Hizo cálculos repetitivos, buscó patrones, escribió fórmulas.

Si hubiera tenido Python, habría automatizado todo eso. Tú vas a aprender a hacer lo que Kepler hacía a mano, pero con código.

Los datos cuentan historias. Python te ayuda a descubrirlas más rápido.
