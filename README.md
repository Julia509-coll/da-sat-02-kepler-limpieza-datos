# 🚀 Proyecto de Análisis: Kepler & Biblioteca

¡Bienvenida a mi repositorio de Python! En este proyecto he trabajado con lógica de programación y limpieza de datos.

## 📂 Contenido del Proyecto

### 1. Limpieza de Datos (Misión Kepler)
En la carpeta `da-sat-02-kepler-limpieza-datos` encontrarás el análisis de una misión espacial.
* **Objetivo:** Limpiar datos de telemetría y sensores.
* **Logros:** Filtrado de valores nulos y corrección de tipos de datos.

### 2. Gestión de Biblioteca
Script interactivo para gestionar préstamos de libros.
* **Funciones:** Cálculo de multas, estadísticas por género literario y control de retrasos.
* **Tecnologías:** Python (Diccionarios, Listas y Random).

## 🛠️ Cómo ejecutarlo
1. Clona este repositorio.
2. Abre la terminal y ejecuta: `python biblioteca.py` o `python ventas.py`.
